# 讀書筆記本系統 - 產品需求文件 (PRD)

## 1. 產品概述 (Product Overview)
**產品名稱**：讀書筆記本
**產品目標**：打造一個專為愛書人與學生設計的數位筆記系統，讓使用者能輕鬆管理書單、追蹤閱讀進度，並透過 Markdown 撰寫結構化、易讀的讀後心得。

## 2. 目標受眾 (Target Audience)
* **大學生與研究生**：需要大量閱讀文獻、參考書，並整理筆記與重點標籤。
* **愛好閱讀者**：喜歡記錄讀書歷程、撰寫讀後感，並為書籍打分數的讀者。

## 3. 核心功能需求 (Core Features)

### 3.1 書籍資訊管理
* **新增書籍**：使用者可輸入書名、作者等基本資訊來建立書籍檔案。
* **刪除書籍**：可移除不再需要追蹤或記錄的書籍。
* **書單總覽**：以列表或網格方式呈現所有已加入的書籍。

### 3.2 閱讀進度追蹤
* **進度記錄方式**：支援兩種模式，可依個人喜好選擇輸入「百分比 (%)」或「目前頁數 / 總頁數」。
* **視覺化進度條**：在書籍總覽或詳細頁面顯示進度條，直觀呈現目前的閱讀狀態。

### 3.3 Markdown 心得編輯器
* **支援 Markdown 語法**：支援標題、粗體、清單、引言、程式碼區塊等基礎 Markdown 語法，讓筆記更具結構性。
* **即時預覽 (可選/未來擴充)**：提供編輯與預覽雙欄模式，或所見即所得的編輯體驗。

### 3.4 書籤功能
* **快速標記**：使用者可以對特定書籍或某篇筆記加入書籤，方便日後快速尋找與回顧。
* **書籤列表**：提供專屬的書籤頁籤或篩選器，集中檢視所有被標記的項目。

### 3.5 筆記分類標籤 (Tags)
* **自訂標籤**：使用者可自由新增標籤（如：`文學`、`技術`、`心理學`、`待二刷` 等）。
* **標籤篩選**：在總覽頁面可透過點擊標籤，快速篩選出對應分類的書籍或筆記。

### 3.6 閱讀心得星級評分
* **五星評分制**：可針對已讀完（或閱讀中）的書籍給予 1 至 5 顆星的評價。
* **評分顯示**：星級評價會顯示於書籍列表與筆記詳細頁面中，作為推薦指數參考。
